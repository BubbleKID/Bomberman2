## Control
1.  Player1 control setting
MoveUp    `W`
MoveLeft    `A`
MoveDown    `S`
MoveRight    `D`
Plant Bomb `Space`
Explode Bomb   `f`

2.  Player2 control setting
MoveUp    `Up`
MoveLeft    `Left`
MoveDown    `Down`
MoveRight    `Right`
Plant Bomb `Enter`
Explode Bomb   `Num0`
